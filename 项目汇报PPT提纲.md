# 中国近五年天气抓取与可视化分析项目汇报PPT提纲

- 项目名称：中国近五年天气抓取与可视化分析
- 汇报人
- 学号 / 班级 / 专业

## 第 2 页：项目背景

- 天气数据具有明显的时间序列与空间分布特征
- 单纯表格难以体现多维天气变化规律
- 需要将抓取、分析和可视化结合起来

可讲重点：

- 为什么要做天气可视化
- 为什么选择近五年作为分析区间

## 第 3 页：项目目标

- 抓取中国主要城市近五年历史天气数据
- 进行清洗、聚合和趋势分析
- 构建交互式可视化大屏
- 支持时间轴地图和省份联动趋势

## 第 4 页：技术路线

- 数据抓取：Open-Meteo Historical Weather API
- 数据处理：pandas + numpy
- 图表生成：pyecharts
- 地图与联动：ECharts JS
- 输出结果：HTML 大屏 + Markdown 报告

## 第 6 页：数据来源与采样范围

## 第 10 页：核心功能一 地图时间轴
- 地图区已可直接触发“更新全部数据”“仅重抓失败数据”“中断任务”
- 数据任务运行中会禁用重复触发按钮，并显示实时进度

- 拖动时间轴查看不同月份
- 支持自动播放
- 支持指标切换
- 展示分省温度、降水、风速
